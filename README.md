# ReceiptKeeper

A Flask-based personal finance tracker where users can add receipts, set monthly budgets per category, track spending with progress bars, export CSV, and receive monthly email reports.

## Features
- User authentication (register/login)
- Add, edit, delete receipts
- Set monthly budgets per category
- Real-time budget tracking with progress bars
- Dark mode
- Search and filter receipts
- CSV export
- Monthly spending chart (Chart.js)
- Automated monthly email reports (SMTP)

## Tech Stack
- Python (Flask)
- SQLite
- HTML/CSS
- Chart.js
- Gmail API (SMTP)

## Deployment
- Deployed on Render / PythonAnywhere
